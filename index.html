Projects

Factorio Server Command Log File Parser

For use in Factorio_Headless server folder on a Ubuntu virtual machine. To start both the log file parser as well as the Factorio server type ./run.sh into the terminal in the folder with the headless server and my scripts. My program reads the factorio log to see if an admin has said a command. My program will activate Bash scripts if commands are detected that will stop the Factorio Server and make command-specific changes to it and then run it again.

Royal Tournament Rank Sign Up Parser and Team Simulation

Imports sign-up from from royal tournament google sheet as csv. Creates several different types of lists that output to outputs folder and cleans improper inputs with regex. Creates lists in order of rank priority with Age of Empires 3 Definitive Edition ranks prioritized over Age Of Empires 3 Complete edition and vise versa. Creates a raw file and cleaned version, raw for viewing, cleaned for parsing for the simulation. Parses through cleaned version with team_simulation.py and creates a file called team_re and team_de that represents team captians and choosing order to see which draft from a specific amount of players seems the most fair and enjoyable for people playing and watching. Teams are in groups of 3 and team captians blindly choose off of rank in the simulation.

Simple Discord Bot

After a user in a discord server with the Antwan Bot writes $hello, the bot simply sends "Hello!" back. This bot is very simple but the more complicated foundation has been laid to enable me to make changes on the top of this to make a more specific Discord Bot.
